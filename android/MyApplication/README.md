# FokusMask

**Обменяй шум на концентрацию**

FokusMask — это приложение для генерации динамического фонового звука, которое помогает маскировать бытовой шум и улучшать концентрацию. В отличие от обычных плееров с зацикленными записями, FokusMask генерирует звук в реальном времени, что исключает эффект повторения и раздражения.

## Проблема

Студенты, удаленные работники и офисные сотрудники вынуждены работать в условиях постоянного фонового шума (соседи, разговоры, бытовая техника). Существующие решения (музыка, ANC-наушники, простые приложения) либо неэффективны против речи, либо сами отвлекают.

## Решение

FokusMask использует принцип звуковой маскировки — добавление контролируемого фонового звука, который делает внешние раздражители менее заметными для мозга. Приложение генерирует "цветные" шумы (белый, розовый, коричневый) с алгоритмом случайных вариаций, создавая эффект живого, неповторяющегося звука.

**Как это работает:**
1. Пользователь выбирает базовый звук
2. Приложение создаёт динамический, неповторяющийся звуковой поток
3. Специально подобранные частоты маскируют внешние раздражители:
   - **Коричневый шум** — заглушает низкочастотный гул и крики
   - **Розовый шум** — маскирует разговоры в офисе
   - **Белый шум с вариациями** — создаёт эффект живого дождя

## Функциональность прототипа (v0.1)

-  Генерация белого, розового и коричневого шума в реальном времени
-  Три готовых пресета: "От соседей", "Офисный режим", "Дождь"
-  Таймер на 25 минут с плавным затуханием звука
-  Регулировка громкости
-  Работа в фоновом режиме
-  Сохранение последних настроек
-  Онбординг при первом запуске (в разработке)

# Клонировать репозиторий
git clone https://github.com/poznik-git/FokusMask.git
cd FokusMask

# Установить зависимости
pip install pyaudio numpy

# Запустить приложение
python main.py

### В разработке:
- **Звуковой движок**: генерация белого, розового и коричневого шума в реальном времени
- **Интерфейс**: главное окно с тремя крупными кнопками-пресетами
- **Пресеты**:
  - «От соседей» — коричневый шум
  - «Офисный режим» — розовый шум
  - «Дождь» — белый шум с вариациями
- **Управление**: кнопка «Стоп» и ползунок громкости
- **Фоновый режим**: звук не прерывается при сворачивании приложения
- **Аудиотека**: подготовлено 6 звуков
- **Дизайн-макеты**: разработаны экраны главного меню, таймера и микшера

## Технологии

- **Язык:** Python 3.9+
- **Интерфейс:** Tkinter
- **Звук:** PyAudio + NumPy
- **Сборка:** PyInstaller (для создания .exe/.app)
